> Мягкий портативный пак для Windows, когда голос Discord, медиа или экран подключения перестают работать из‑за DPI-фильтрации.

## С чем помогает этот пак

- Discord зависает на подключении
- Голосовой чат не подключается или обрывается
- Циклы переподключения RTC
- Проблемы загрузки медиа/CDN
- Стабильный запуск через **`zapret-discord.bat`**

## Скачивание

Откройте последний релиз и скачайте архив:

```text
Releases -> Latest -> скачать fixdiscord.rar
```

После распаковки запустите:

```text
zapret-discord.bat
```

Также доступны отдельные лаунчеры по сервисам: `zapret-discord.bat`, `zapret-youtube.bat`, `zapret-telegram.bat`, `zapret-dpi.bat`, `zapret-все-сервисы.bat`.

Пакетный файл автоматически запускает нужный исполняемый файл — вручную выбирать файлы не нужно.
